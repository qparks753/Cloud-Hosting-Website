Hayai CLI

Hayai CLI allows you to interact with the Hayai API via the command line. It supports most functionality found in the control panel.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install Hayai-cli

NPM

$ npm install Hayai-cli

Yarn

$ yarn install Hayai-cli
